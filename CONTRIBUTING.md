# Contributing to Cartographer Drones

Thanks for your interest in contributing!

## Reporting Issues
Please use [GitHub Issues](https://github.com/jwhicks727/cartographer-drones/issues) to report bugs or request features. Check existing issues before opening a new one.

## Pull Requests
- Fork the repo and create a branch from main
- Follow existing code style and conventions
- Update CHANGELOG.md under [Unreleased] with a summary of your changes
- Open a PR against main with a clear description of what you changed and why

## Development Setup
1. Install Java 25
2. Clone the repo
3. Run .\gradlew build to confirm everything compiles

## Questions
Join the [Discord](https://discord.gg/yourinvitecode) for discussion.
